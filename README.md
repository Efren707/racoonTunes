# RacoonTunes
[RacoonTunes](https://racoontunes.herokuapp.com/#/) is a soundcloud clone, it is an audio distribution platform and music sharing website that allows users to upload, share, and stream music.

## Technologies
- React
- Redux
- JavaScript
- Rails
- Postgresql
- CSS
- AWS
